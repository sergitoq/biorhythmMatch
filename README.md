# BiorhythmMatch

Interactive biorhythm calculator with compatibility analysis for two people.

Tracks physical, emotional and intellectual cycles with comparative charts and a 46-day window. Features saved profiles and compatibility scoring.

## Live App

https://sergitoq.github.io/biorhythmMatch/

## Features

- Biorhythm cycles: Physical (23d), Emotional (28d), Intellectual (33d)
- Passion chart combining physical + emotional cycles
- Compatibility analysis between two people
- Saved profiles via localStorage
- Dark theme, mobile friendly
